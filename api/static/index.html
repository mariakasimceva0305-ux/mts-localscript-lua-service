            Справка

            Режимы
            Полоса под шапкой: новый скрипт, правка, проверка. «Быстро» / «Глубоко» — в «Глубоко» к /generate и /edit добавляется include_diagnostics: true (расширенный JSON и блок диагностики под сводкой).
            Профиль и настройки
            «Гость» / «Локальный» — в шапке. «Платформа» (⋯) — тема, масштаб, шрифт кода, история, параметры полоски над чатом.
            Octapi в коде
            Доступ к полям схемы: wf.vars.<имя>. Стартовые данные сценария — wf.initVariables (см. документацию MWS Octapi).
            Запросы к API
            Новый скрипт: текст из чата, код редактора не отправляется — POST /generate. Правка: инструкция + код редактора — POST /edit. Проверка: только код редактора — POST /validate.
            Результат
            Блок «Результат»: сводка, вкладка проверки, чанки знаний, пояснение модели.
            Валидация Lua
            Жёсткие ошибки не дают завершить проверку успехом; область редактора кратко подсвечивается.
            Контур
            Инференс через Ollama в контуре организации; в этом решении не вызываются внешние облачные LLM API.

            Платформа

            • API и Ollama в контуре организации
            • Внешние облачные LLM не используются
            • Проверка: luac и правила Octapi
            • Сводка ответа: полоса над вкладками; расширенная диагностика — раскрывающийся блок ниже
